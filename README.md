# cr_mobile_menu
A mobile menu project for an existing portifolio project.

## Built With

- HTML
- CSS
- JavaScript


## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites
- Npm installed
- Editor such as VSCode
- Browser

### Setup
- clone the repo

### Install
- Run `npm i`

### Usage
- Open `index.html` file to view app
